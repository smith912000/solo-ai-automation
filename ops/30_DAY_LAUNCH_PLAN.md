# 30-Day Launch Plan

> From zero to first paying client in 30 days.

---

## Week 1: Infrastructure

**Goal:** All systems operational, ready to demo.

### Day 1-2: Core Setup

- [ ] Create Supabase project
- [ ] Run schema.sql
- [ ] Verify tables created
- [ ] Set up n8n instance (Railway or Fly.io)
- [ ] Import workflow from JSON
- [ ] Test webhook endpoint responds

### Day 3-4: Agent Deployment

- [ ] Create Python virtual environment
- [ ] Install dependencies
- [ ] Configure .env with API keys
- [ ] Test agent locally with sample data
- [ ] Deploy agent as HTTP endpoint (Railway/Fly.io)
- [ ] Connect n8n to agent endpoint

### Day 5: Integration

- [ ] Connect n8n to Supabase
- [ ] Set up Slack notifications
- [ ] Configure SendGrid (verify domain)
- [ ] Test full flow: webhook → qualify → email queue

### Day 6-7: Observability

- [ ] Set up LangSmith
- [ ] Configure cost tracking
- [ ] Create monitoring alerts
- [ ] Test kill switches work
- [ ] Document in runbook

**Week 1 Deliverable:** Working demo you can show.

---

## Week 2: Build Your Demo

**Goal:** One polished demo automation you can show prospects.

### Day 8-9: Perfect the Flow

- [ ] Run 20+ test leads through system
- [ ] Review qualification accuracy
- [ ] Tune rubric/scoring
- [ ] Improve email templates
- [ ] Fix any edge cases

### Day 10-11: Case Study Data

- [ ] Process 50 sample leads
- [ ] Calculate average:
  - Processing time
  - Token cost per lead
  - Qualification distribution
- [ ] Screenshot example outputs
- [ ] Record 2-min demo video (Loom)

### Day 12-13: Documentation

- [ ] Finalize one-pager with real metrics
- [ ] Update pricing based on actual costs
- [ ] Create simple landing page (Carrd or Notion)
- [ ] Prepare discovery call script

### Day 14: Dry Run

- [ ] Do a mock discovery call (with a friend)
- [ ] Walk through full demo
- [ ] Practice explaining value prop
- [ ] Refine based on feedback

**Week 2 Deliverable:** Demo-ready with collateral.

---

## Week 3: Package It

**Goal:** Everything ready for client #1.

### Day 15-16: Legal & Business

- [ ] Finalize Terms of Service
- [ ] Prepare contract template
- [ ] Set up payment system (Stripe)
- [ ] Create client intake form

### Day 17-18: Onboarding System

- [ ] Test onboarding checklist end-to-end
- [ ] Create client welcome email template
- [ ] Set up client Slack channel template
- [ ] Prepare training materials

### Day 19-20: Template Everything

- [ ] Create rubric variants for 3 industries
- [ ] Prepare email tone variants
- [ ] Document configuration options
- [ ] Build "quick start" guide for clients

### Day 21: Final Polish

- [ ] Review all documentation
- [ ] Test complete flow one more time
- [ ] Update landing page with final copy
- [ ] Set up calendar booking (Calendly)

**Week 3 Deliverable:** Turn-key package ready to sell.

---

## Week 4: Get Client #1

**Goal:** Signed contract and automation live.

### Day 22-23: Outreach

- [ ] List 20 ideal prospects
- [ ] Send personalized outreach
- [ ] Post on LinkedIn (share what you built)
- [ ] Reach out to network

### Day 24-25: Discovery Calls

- [ ] Book at least 3 calls
- [ ] Run discovery process
- [ ] Identify best-fit prospect
- [ ] Send proposal same day

### Day 26-27: Close & Onboard

- [ ] Follow up on proposals
- [ ] Negotiate if needed
- [ ] Sign contract
- [ ] Collect first payment
- [ ] Begin onboarding

### Day 28-30: Deliver

- [ ] Configure for client #1
- [ ] Run in approval mode
- [ ] Train client on queue
- [ ] Get first qualified leads through
- [ ] Collect feedback
- [ ] Get testimonial (even informal)

**Week 4 Deliverable:** First paying client live.

## Contingency Plans

### If infrastructure takes longer (Week 1)

- Extend by 2-3 days
- Compress Week 3 (template less)
- Keep Week 4 dates

### If no discovery calls by Day 25

- Increase outreach volume 3x
- Offer free trial to first client
- Ask network for warm intros
- Post more content

### If prospect says no

- Ask: "What would need to be true for this to work?"
- Offer pilot at reduced rate
- Move to next prospect fast
- Don't get stuck on one lead

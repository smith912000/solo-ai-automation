# Voice AI Agent (Calls)

> **Purpose**: Place and manage AI-driven calls for outreach or qualification.

## Scope

- Outbound calls to warm/cold leads
- Call summaries and outcomes
- Handoff to human when needed

## Integration

This agent is a stub. Wire it to a voice provider such as Vapi, Bland.ai,
or a custom Twilio pipeline.

## Files

| File | Purpose |
|------|---------|
| `agent/main.py` | Provider-agnostic call interface |
